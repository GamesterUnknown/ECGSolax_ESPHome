# ECGSolax_ESPHome
 ESPHome example configuration to monitor and control a ECGSolax 6.2kWt inverter via RS232

# Requirements
 ESPHome 2024.6.0 or higher.
 One half of an ethernet cable with RJ45 connector
 RS232-to-TTL module (MAX3232CSE f.e.)
 Generic ESP32
